# Deckbuilder Prototype (Godot 4)

This project is a football-themed deckbuilder inspired by Slay the Spire, focused on Brazilian football of the 80s. All code and documentation are in English for learning purposes; user interfaces are in Portuguese (BR).

## Project Structure
- `scenes/` — Godot scenes (UI, gameplay, etc.)
- `scripts/` — GDScript files
- `assets/` — Art, audio, and other assets
- `docs/` — Design, research, and planning documents

## How to Run
1. Open the project folder in Godot 4.
2. Run the main scene (to be defined).
